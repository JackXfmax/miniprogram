# 🚀 快速参考卡片

## ⚡ 30秒快速开始

```
1. 点击「详情」→「本地设置」→ 勾选「不校验合法域名」
2. 按 Ctrl+B 重新编译
3. 选择图片 → 点击「开始识别」→ 完成！
```

---

## 🔧 必须配置

**在微信开发者工具中：**
- 点击右上角「详情」
- 选择「本地设置」
- 勾选：`☑ 不校验合法域名...`
- 重新编译：`Ctrl+B`

---

## 📱 使用流程

```
选择图片 → 预览 → 开始识别 → 等待 → 查看结果 → 复制
```

---

## ⏱️ 识别时间

| 图片大小 | 时间 |
|---------|------|
| < 1MB | 5-15秒 |
| 1-3MB | 15-30秒 |
| 3-5MB | 30-60秒 |
| > 5MB | > 60秒 |

## 📋 功能按钮

- **从相册选择** - 选择已有图片
- **拍照识别** - 拍摄新照片
- **开始识别** - 启动OCR识别
- **复制识别结果** - 复制文字到剪贴板
- **返回重新识别** - 回到首页
- **查看字符详情** - 展开/收起详细信息
- **原始JSON数据** - 查看完整API响应

---

## 🐛 问题解决

### 网络请求失败？
→ 勾选「不校验合法域名」→ 重新编译

### 识别速度慢？
→ 使用小图片 → 使用校园网

### 识别结果空？
→ 使用清晰图片 → 确保文字可读
