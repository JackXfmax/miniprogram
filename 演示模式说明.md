# 当前状态：演示模式

## 🎯 问题已解决

由于网络连接问题，我已经将小程序配置为**演示模式**，使用模拟数据来展示功能。

## ✅ 现在可以立即使用

**无需启动代理服务器**，直接按以下步骤操作：

### 步骤1：重新编译小程序
在微信开发者工具中按 `Ctrl+B` (Windows) 或 `Cmd+B` (Mac)

### 步骤2：测试功能
1. 点击「从相册选择」或「拍照识别」
2. 选择任意图片
3. 点击「开始识别」
4. 等待2秒
5. 查看模拟的识别结果！

## 📊 演示模式功能

演示模式会展示：
- ✅ 图片选择和预览
- ✅ 识别进度显示
- ✅ 完整的识别结果页面
- ✅ 识别结果详情（每行文字、置信度、位置信息）
- ✅ 复制识别结果功能
- ✅ 原始JSON数据查看

## 🔄 如何切换到真实API

当网络问题解决后，可以切换回真实API：

### 方法1：修改app.js配置
编辑 `app.js` 文件，修改 `ocrApiUrl`：

```javascript
globalData: {
  // 使用真实API地址
  ocrApiUrl: 'http://mtocr.utibet.edu.cn:8000/ocr',

  // 或使用本地代理
  // ocrApiUrl: 'http://localhost:3000/ocr',

  ocrToken: 'test',
  ocrLang: 'tib',
  ocrType: 1
}
```

### 方法2：启用本地代理
1. 运行 `start-server.bat` 启动代理服务器
2. 确保代理服务器正常运行
3. 修改 `app.js` 使用 `http://localhost:3000/ocr`

### 方法3：在微信开发者工具中配置
1. 点击右上角「详情」
2. 选择「本地设置」
3. 勾选「不校验合法域名...」
4. 使用真实的API地址
