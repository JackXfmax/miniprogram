## 🚀 3步开始使用

### 第1步：配置开发者工具（必须！）

**在微信开发者工具中：**

1. 点击右上角「详情」按钮
2. 选择「本地设置」选项卡
3. 勾选「不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书」
4. 关闭设置窗口

### 第2步：重新编译

按 `Ctrl+B` (Windows) 或 `Cmd+B` (Mac)

或点击顶部工具栏的「编译」按钮

### 第3步：开始识别

1. 点击「从相册选择」或「拍照识别」
2. 选择包含文字的图片
3. 点击「开始识别」
4. 等待识别完成
5. 查看结果！

## ⏱️ 识别时间说明

| 图片大小 | 预计时间 | 说明 |
|---------|---------|------|
| < 1MB | 5-15秒 | 较快 |
| 1-3MB | 15-30秒 | 适中 |
| 3-5MB | 30-60秒 | 较慢 |
| > 5MB | > 60秒 | 建议压缩 |

**提示：**
- 首次识别可能较慢
- 网络状况影响速度
- 图片越大越慢

## 🎯 识别结果说明

识别完成后，您会看到：

### 1. 整体信息
- **识别耗时**：API处理时间
- **整体置信度**：识别准确率（0-1之间）
- **识别行数**：识别的文字行数

### 2. 每行文字详情
- **识别文字**：识别出的文字内容
- **置信度**：该行文字的准确率
- **位置坐标**：文字在图片中的位置（x, y）
- **尺寸**：文字区域的宽度和高度
- **字体**：识别出的字体类型

### 3. 字符详情（可展开）
- 点击「查看字符详情」
- 查看每个字符
- 查看每个字符的置信度
- 查看每个字符的位置

### 4. 原始JSON数据
- 点击「原始JSON数据」
- 查看完整的API响应
- 用于调试或导出数据

## 🐛 常见问题解决

### Q1: 还是提示"网络请求失败"

**解决方案：**
1. 确认已勾选「不校验合法域名」
2. 确认已重新编译（Ctrl+B）
3. 检查电脑网络连接
4. 尝试使用校园网

### Q2: 识别结果为空

**原因：**
- 图片中没有文字
- 图片太模糊
- 文字太小

**解决：**
- 使用更清晰的图片
- 确保文字足够大
- 尝试其他图片

### Q3: 识别速度很慢

**原因：**
- 图片太大
- 网络慢
- 服务器繁忙

**解决：**
- 压缩图片（< 5MB）
- 使用更快的网络
- 使用校园网地址
