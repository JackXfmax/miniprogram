# 快速配置指南

## ⚠️ 必须配置：关闭HTTPS校验

由于OCR API使用HTTP协议，需要在微信开发者工具中关闭HTTPS校验。

### 配置步骤（1分钟完成）：

#### 第1步：打开详情设置
在微信开发者工具中，点击右上角的「详情」按钮

#### 第2步：进入本地设置
在弹出的窗口中，选择「本地设置」选项卡

#### 第3步：勾选不校验选项
找到「本地设置」区域，勾选：
```
☑ 不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书
```

#### 第4步：重新编译
按 `Ctrl+B` (Windows) 或 `Cmd+B` (Mac) 重新编译小程序

---

## ✅ 配置完成！

现在小程序可以直接使用了：

1. **选择图片**：点击「从相册选择」或「拍照识别」
2. **开始识别**：点击「开始识别」
3. **等待结果**：首次识别可能需要较长时间（图片越大越慢）
4. **查看结果**：识别完成后自动跳转到结果页面

---

## 🔧 API地址配置

当前使用外网地址：`http://mtocr.utibet.edu.cn:8000/ocr`

**如果您在校园网内**，可以修改为更快的校园网地址：

编辑 `app.js` 文件，修改 `ocrApiUrl`：

```javascript
// 外网地址（当前）
ocrApiUrl: 'http://mtocr.utibet.edu.cn:8000/ocr'

// 校园网地址（更快）
// ocrApiUrl: 'http://222.19.82.145:8000/ocr'
```

## 📱 使用流程

### 完整使用流程：

1. **配置开发者工具**
   - 勾选「不校验合法域名」
   - 重新编译

2. **选择图片**
   - 点击「从相册选择」
   - 或点击「拍照识别」

3. **预览图片**
   - 查看图片预览
   - 查看图片大小

4. **开始识别**
   - 点击「开始识别」
   - 等待识别完成（可能需要几秒到几十秒）

5. **查看结果**
   - 自动跳转到结果页面
   - 查看识别的文字
   - 查看每个字符的详情

6. **使用结果**
   - 点击「复制识别结果」
   - 或查看「原始JSON数据」

## 🎯 常见问题

### Q1: 还是提示"网络请求失败"

**解决方法：**
1. 确认已勾选「不校验合法域名」
2. 确认已重新编译（Ctrl+B）
3. 检查网络连接是否正常
4. 尝试使用校园网地址

### Q2: 识别速度很慢

**原因：**
- 图片越大，识别越慢
- 首次连接可能较慢
- 网络状况影响速度

**解决方法：**
- 使用较小的图片（< 5MB）
- 使用校园网地址
- 耐心等待

### Q3: 识别结果不准确

**原因：**
- 图片不清晰
- 文字太小
- 图片模糊

**解决方法：**
- 使用高清晰度图片
- 确保文字清晰可读
- 避免图片过度压缩

### Q4: 如何修改语言设置？

编辑 `app.js` 文件，修改 `ocrLang`：

```javascript
ocrLang: 'tib'  // 藏文
ocrLang: 'chi'  // 中文
ocrLang: 'eng'  // 英文
```

## 💡 优化建议

### 1. 选择合适的图片
- 清晰度高的图片
- 文字大小适中
- 文件大小适中（< 5MB）

### 2. 使用校园网（如果可用）
- 速度更快
- 更稳定

### 3. 耐心等待
- 首次识别可能较慢
- 图片越大越慢
- 网络状况影响速度
