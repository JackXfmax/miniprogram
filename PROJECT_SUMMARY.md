# 藏文OCR识别小程序 - 项目总结

## 项目概述

基于微信小程序的藏文OCR识别应用，调用西藏民族大学OCR API实现图片文字识别功能。

## 核心功能

### 1. 图片采集
- 从相册选择图片
- 拍照识别
- 图片预览

### 2. OCR识别
- 调用远程API进行文字识别
- 支持藏文、中文、英文混合识别
- 识别进度显示

### 3. 结果展示
- 识别结果列表
- 每行文字的置信度
- 字符位置信息
- 字符详情展开

### 4. 结果处理
- 一键复制识别结果
- 原始JSON数据查看
- 返回重新识别

## 技术架构

### 前端
- 微信小程序原生框架
- WXML + WXSS + JavaScript
- 响应式设计

### 后端API
- 西藏民族大学OCR服务
- RESTful API设计
- JSON数据格式

### 数据流
```
用户选择图片 → 图片转Base64 → 调用OCR API → 解析结果 → 展示识别结果
```

## API接口说明

### 请求地址
- 校园网：`http://222.19.82.145:8000/ocr`
- 外网：`http://mtocr.utibet.edu.cn:8000/ocr`

### 请求方式
- Method: POST
- Content-Type: application/json

### 请求参数
```json
{
  "image": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...",
  "lang": "tib",
  "type": 1,
  "token": "test"
}
```

### 响应示例
```json
{
  "msg": "success",
  "error": 0,
  "time": 2.36,
  "score": 0.9607,
  "result": [
    {
      "word": "藏文文字",
      "score": 0.99,
      "x": 100,
      "y": 200,
      "w": 300,
      "h": 50,
      "fm": "宋体",
      "index": 1,
      "words": [...]
    }
  ]
}
```
