# 快速启动指南

## 第一步：启动代理服务器

**方法一：双击启动（推荐）**
1. 在 `miniprogram` 文件夹中找到 `start-server.bat`
2. 双击运行
3. 看到 "✓ OCR代理服务器已启动" 表示成功

**方法二：手动启动**
1. 打开命令提示符（CMD）或 PowerShell
2. 进入 `miniprogram` 文件夹
3. 运行命令：`node proxy-server.js`

## 第二步：重新编译小程序

1. 在微信开发者工具中
2. 按 `Ctrl + B` (Windows) 或 `Cmd + B` (Mac) 重新编译
3. 或者点击「编译」按钮

## 第三步：测试识别功能

1. 点击「从相册选择」或「拍照识别」
2. 选择一张包含文字的图片
3. 查看图片预览和大小信息
4. 点击「开始识别」
5. 等待识别完成

## 常见问题解决

### Q1: 提示"网络请求失败"
**解决方案：**
- 确保代理服务器已启动（看到"✓ OCR代理服务器已启动"）
- 检查是否显示"本地地址: http://localhost:3000"

### Q2: 提示"图片base64编码无效"
**解决方案：**
- 重新编译小程序（Ctrl+B）
- 尝试选择其他图片
- 确保图片格式是 JPG 或 PNG

### Q3: 代理服务器启动失败
**解决方案：**
- 确保已安装 Node.js（下载地址：https://nodejs.org/）
- 检查端口 3000 是否被占用
- 尝试关闭其他程序后重试

### Q4: 识别结果为空
**解决方案：**
- 确保图片清晰
- 尝试使用包含更多文字的图片
- 检查控制台日志查看详细错误信息

## 调试信息

小程序首页会显示：
- API地址：当前使用的OCR API地址
- 语言设置：当前的识别语言
- 图片大小：选择的图片文件大小

## 文件说明

- `start-server.bat` - 一键启动代理服务器
- `proxy-server.js` - 代理服务器程序
- `app.js` - 小程序配置（可修改API地址）
